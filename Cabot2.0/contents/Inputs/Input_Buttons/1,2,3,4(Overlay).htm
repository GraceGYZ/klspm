• Click the number 1/2/3/4 in the input source window to overlay the current input source to the screen for preview or live.

  • Left-click the number, the green is the superimposed upper screen to the preview (PVM) screen, left-click the lower screen again,      

Right-click the number, the red is superimposed on the upper screen to the live broadcast (PGM) screen, right-click the lower screen again,When superimposed to PVW/PGM at the same time, the button is displayed in blue.

  • In addition to the current input source, the PVW/PGM window can superimpose up to 4 button contents, and each layer can only have one input source The PVM/PGM window switches the input source, and the overlay content and effects are still there.

  • In the Switch Control window, directly left/right click on 1/2/3/4 to superimpose the last input source content to PVW/PGM. If there is no record, there is no response after clicking the button.

  • Click to open the overlay settings, you can set the overlay type, transition effect and transition duration for level 1/2/3/4 respectively

  1. Type

    Full screen：Overwrite the current input source.

    Pinp：Displayed on the input source screen at the specified screen size and position. Support manual modification of parameters to adjust the position, crop, and zoom the screen size. The blue screen is to adjust the preview effect.

  2. Trans effect

    Supports the option to specify the transition effect of the superimposed upper/lower screen.

    If the overlay content is a subtitle template, the effect is that the transition of the overlay setting and the input and output animation of the subtitle template work together.

  3. Length

    Specifies the transition effect duration in milliseconds.
