•  File

  • New：支持新建一个空白模板进行自定义编辑。
  • CG：打开字幕列表，支持添加在线字幕模板、本地模板进行自定义。
  • Pinp：打开画中画列表，支持添加在线画中画模板、本地画中画模板进行自定义。
  • 保存：保存当前字幕模板到预制模板的用户目录下，若未保存编辑模板进行新建、导入、退出操作会出现系统提示是否保存当前模板。
  • 另存为：另存当前字幕模板到预制模板的用户目录下。
  • 导入：支持直接选择本地路径下字幕模板。
  • 导出：支持将当前模板导出到本地路径。
  • 关闭：关闭字幕编辑器，未保存的字幕模板系统会提示是否保存。
  • 编辑

  • 用户自定义编辑界面，支持undo/redo、剪切、复制、粘贴、及删除等基本操作

  • 支持每层物件的置顶/置底，上移/下移

  • 支持添加字幕物件包括：文本，图片，序列，视频，时钟，滚屏
  • 文本编辑

  1. 支持调整位置、旋转
  2. 支持更改字体、大小等文字相关参数
  3. 支持修改面、边、阴影颜色
  • 图片编辑

  1. 支持调整位置、旋转
  2. 支持更改图片路径
  3. 支持整体缩放、裁剪
  • 视频编辑

  1. 支持调整位置、旋转
  2. 支持更换视频输入源
  3. 支持整体缩放、裁剪
  • 序列编辑

  1. 支持调整位置、旋转
  2. 支持更改文件路径
  3. 支持整体缩放、裁剪
  4. 支持设置是否循环播放
  • 时钟编辑

  1. 支持调整位置、旋转
  2. 支持设置时钟类型、显示格式、起始时间
  • 滚屏编辑

  1. 支持设置滚屏文件、背景颜色、滚动类型等参数
  2. 支持更改字体、大小等文字相关参数
  • 转场

支持设置字幕转场入/出特效及效果推迟、持续时间

  • 数据

设置字幕文件为多行可替换项

  • 通用替换信息属性

  • 支持设置物件在字幕属性窗口显示的可替换名称
  • 支持将物件设置为可替换项显示在字幕属性窗口
