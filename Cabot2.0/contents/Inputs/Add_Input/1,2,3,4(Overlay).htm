• Mobile input source can be added. After logging in, Cabot will enable "Allow others/devices to connect" by default and display QR code
  • Using the iCabot APP (the iCabot APP can be downloaded by clicking on the Cabot ), users can scan the QR code to connect to Cabot, and access information will be displayed on Cabot
  • After the mobile phone is connected, it can display the mobile phone screen (screen recording live) or the content captured by the mobile phone camera (camera live).

  • iCabot
  1. Cabot's mobile terminal live helper , it' s convenient for users to quickly access mobile phone video and audio signals as input source to the Cabot
  2. After downloading the iCabot APP on mobile phone, can login by wechat / account.

 

         3. After login, the user can click "Scan" or "enter the program number" to connect to the target Cabot and select "camera/screen recording" as the content to live.

  1. Select screen recording. All content on the phone screen, including notifications, will be recorded. Enable Do Not Disturb mode to avoid unexpected notifications
  2. Select camera live and support Mic, lens flip and beauty operation;

 

        4. The selected output content will be directly sent to Cabot, and the Cabot can show the mobile input source connection instantly

        5. After the APP is closed, Cabot will be displayed as disconnected.also ,after Cabot deletes the input source, the APP will prompt "Remote connection closed!"
