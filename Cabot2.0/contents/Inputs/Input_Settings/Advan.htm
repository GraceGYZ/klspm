Supports configuring streaming delay in milliseconds to reduce jitter when receiving high-latency network streams.

Support to adjust the image rotation angle

  • Video/Audio File Source Advanced Settings

Supports selecting the playback mode of switching in/out of the live screen.

1. Automatic playback when the input source is cut into the live screen: When the input source is cut into the live screen, the pause state is automatically cut into the playback state.

    Continue playing: When you load, start playing from the current position of the file.

    Play from the beginning: When cutting in, play from the file header (or entry point).

2. Automatically stop when the live screen is cut out: When the input source cuts out the live screen, it automatically stops playing.
