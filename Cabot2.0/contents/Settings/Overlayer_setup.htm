• Click to open the overlay settings, you can set the overlay type, transition effect and transition duration for level 1/2/3/4 respectively

  1. Type

    Full screen：Overwrite the current input source.

    Pinp：Displayed on the input source screen at the specified screen size and position. Support manual modification of parameters to adjust the position, crop, and zoom the screen size. The blue screen is to adjust the preview effect.

  2. Trans effect

    Supports the option to specify the transition effect of the superimposed upper/lower screen.

    If the overlay content is a subtitle template, the effect is that the transition of the overlay setting and the input and output animation of the subtitle template work together.

  3. Length

    Specifies the transition effect duration in milliseconds.
