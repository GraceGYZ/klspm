视频输入源属性窗口如下，

  • 支持设置视频源循环播放
  • 支持进度条控制、游标拖拽播放预览视频文件
  1. POS：当前播放的游标位置，IN：入点位置，OUT：出点位置，DUR：入出点时长。
  2. 视频文件默认播放入点在起点，默认出点在播放结尾位置。
  3. 视频文件非播放状态下时，允许拖拽游标至任意位置，点击播放时从游标点开始播放；若游标位置设置超出入出点范围，则从默认入点开始播放；
  4. 文件处于播放状态下，属性窗口游标禁止拖拽。
  5. 支持按钮操作：打入点/出点、取消入点/出点、播放/入出点播放
  • 若视频输入源已上屏PVW/PGM，属性窗口预览的播放进度与PVW/PGM保持一致。
