字幕属性窗口如下，

  • 支持实时更新，选中实时更新按钮可实现属性窗口的修改内容直接上屏到PVW/PGM
  • 属性窗口以表格形式展示标准可替换列表(2列)或多行可替换列表(多列)：
  1. 标准可替换列表按照1:3宽度比展示物件名称与可替换内容；
  2. 多行可替换列表根据列的个数等分显示区域
  • 窗口的显示内容为字幕编辑器中设置的可替换物件或预制字幕模板中的可替换物件，支持的可替换项包括
  1. 文本：通过窗口输入框直接输入
  2. 图片：通过窗口选择按钮直接替换
  • 可替换列表支持通过拖拽移动行交换顺序
  • 可替换列表支持通过点击勾选按钮控制某行上屏显示

  • 支持通过表格左侧预留按钮区的右键菜单实现可替换列表可添加删除某行
