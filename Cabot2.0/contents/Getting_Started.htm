A screenshot of Cabot2.0's overall user interface is shown below ,

It is mainly divided into the following parts:

  1. Top Menu

The system operation menu bar can set and save system items, add input sources, output settings, add virtual cameras, open various operation windows, system general settings, and also include system help center and other related information.

  • Preview window

The preview window (PVM) of the input source that you want to display to the live screen, the window status is displayed in green, and you can switch to the live screen through the director control.

  • Live window

The live screen (PGM) of the push stream, the window status is displayed in red.

Push streaming recording also uses this window.

  • Switch Control window

PVM/PGM switching operation window, click to achieve preview and live screen switching, and you can set different switching effects, transition durations and overlay effects.

  • Output Control window

In the live broadcast output and recording operation window, after completing the settings, you can click the button to start streaming and live broadcast screen recording. After opening, the duration will be displayed respectively.

A virtual output button is additionally displayed after adding a virtual camera.

  • Input Source window

Each added input source will display a small real-time preview window, which can perform basic operations on the input source. Different types of input sources also include different characteristic property settings.

Left-click on the small window to display the preview window, and right-click to quickly switch to the live broadcast window.

The colored labels indicate the classification of different input sources. Click and drag the small input source window to enter the colored labels to realize the classification display of input sources.

  • Input source operation window

Click the input source small window button to trigger the opening of the input source operation window, which can be used for input source settings, mixing output settings, subtitle attribute settings, and PTZ operations.

  • Bottom bar

Displays the project format and the memory and CPU occupied by the system.
